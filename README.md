# cpp-greatest-of-three-numbers
A simple C++ program that finds the greatest number among three user-entered numbers using if-else statements.

# C++ Greatest of Three Numbers

## Description
This project finds the largest number among three numbers entered by the user.

## Concepts Used
- C++
- User Input
- if-else if-else
- Logical Operators (&&)

## Example

Input:
10
20
30

Output:
C is greatest

Input:
50
20
30

Output:
A is greater

Input:
10
60
30

Output:
B is greater
